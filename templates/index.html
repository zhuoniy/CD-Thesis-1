Interactive Data Science Project Exploratory Analysis Byte4 by Zhuoni

This is the Exploratory Analysis Byte4 of the Interactive Data Science class.

Question: What are locations I offen visit? Where do I like to use/play my mobile phone?

In this project, I recorded my mobile location and mobile screen and battery level within 10 days. Mobile location is recorded as latitude and longitude. Mobile screen usage is saved as true(lighting) or false(sleep). And battery level is scaled to 100% percentage. Based on screen usage and slope of battery level, I can see my mobile phone using frequency. In the visualization, it's easy to see my tracks in these days, and to see my mobile phone usage based on the timeline. In addition, all the visualization sections are interactive, so you can use any part as a filter. So, you can filter the data in one day or several days, or in a large or small region.

Some designed implementation process: when connecting location, battery, screen tables, I should set the connection filter as "tableA.time=tableB.time". But in fact, data in those tables are not collected at the same time, so in Tableau there is no "tableA.time=tableB.time". My method is to segment time for every 15 minutes, time in a same period will be equal. For example, "2018/10/08 08:01" is "equal" to "2018/10/18 08:08". In this way, tables can be connected.

I put battery level together with screen usage, so they result in a good comparison. I use two sections to visualize them in hours and seconds, by using two different visualizing forms.

As an outcome, I can find it's interesting that I offen use phones in the way from home to campus or the way go back home.
